# Security Policy

## Reporting a Vulnerability

We take the security of CAB seriously. If you find a security vulnerability, please do not report it publicly (e.g., via GitHub Issues). Instead, please report it to the maintainer:

- **Email:** kotlin@qq.com

Please include a detailed description of the vulnerability, steps to reproduce it, and any potential impact. We will make every effort to acknowledge your report within 48 hours and provide a timeline for addressing the issue.

Thank you for helping keep CAB secure!
